# ostoo

## Requirements

1. rust dev environment (rustc, cargo, rustup)
2. qemu

### Rustup Components

Install with `rustup component add <component>`:

1. rust-src
2. llvm-tools-preview

### Cargo packages

Install with `cargo install <crate>`:

1. cargo-xbuild
2. bootimage
